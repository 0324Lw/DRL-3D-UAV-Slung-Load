# 🚀 DRL-3D-UAV-Slung-Load

> 基于 PPO (Proximal Policy Optimization) 深度强化学习的三维带载无人机 (UAV with Slung Load) 路径规划与非线性耦合控制算法实现。
> 本项目构建了一个高度自定义、无外部昂贵物理引擎依赖的 3D `Gymnasium` 仿真环境。创新性地引入基于弹簧阻尼模型的柔性绳索欠驱动力学建模、3D 射线投射感知 (Raycasting LiDAR)，以及自适应高度奖励机制，并通过严密的三阶段课程学习 (Curriculum Learning) 彻底解决强化学习中因载荷扰动和高度惩罚导致的“原地悬停”与“坠地自杀”等局部最优难题。

## ✨ 核心特性 (Features)

* 🌍 **高性能纯 Numpy 三维连续物理引擎**：构建 $50m \times 50m \times 20m$ 的三维连续空间，支持动态随机生成且具有“最小净间距”绝对可通过保障的静态圆柱/方块障碍物阵列。
* 🔗 **柔性绳索与耦合非线性动力学**：摒弃简单的刚性杆假设，环境内置高精度的弹簧阻尼模型 (Spring-Damper Model)。真实模拟绳索从“松弛”到“瞬间拉直”的冲击力，以及货物摆动对无人机本体产生的反作用力扰动。
* 📡 **2.5D/3D LiDAR 射线感知模型**：无人机搭载 24 线虚拟激光雷达，在复杂三维障碍物林中进行实时射线投射，返回归一化触碰距离，赋予智能体真实的局部环境感知能力。
* 🎓 **三阶段渐进式课程学习 (Curriculum Learning)**：针对欠驱动系统极高的探索难度，设计“无障碍起降与平稳拉直”、“稀疏障碍物避障”、“密集极限穿梭”三阶段课程，平滑过渡任务难度，保障策略稳健收敛。
* 🛡️ **动态自适应期望高度 (Adaptive Altitude Target)**：首创距离驱动的高度奖励自适应机制，完美消除“保持巡航高度”与“到达终点必须降落”之间的奖励冲突，杜绝 Reward Hacking 现象。

## ⚙️ 柔性载荷耦合动力学建模 (Slung Load Dynamics)

本环境的核心难点在于无人机与下方悬吊质量的非线性耦合。我们对无人机 ($m_q$)、点质量货物 ($m_p$) 以及原长为 $l_0$ 的柔性绳索进行精细建模，仿真步长精细至 $0.01s$。

**1. 弹簧阻尼拉力模型 (Spring-Damper Tension)**
设无人机与货物的实际距离为 $d$，相对速度在绳索方向的投影为 $\dot{d}$。当绳索拉直 ($d > l_0$) 时，基于胡克定律计算标量拉力 $T$：
$T = \max(0, K_s (d - l_0) + K_d \dot{d})$
其中 $K_s$ 为弹性刚度，$K_d$ 为阻尼系数。产生的方向向量 $\mathbf{n}$ 指向无人机。

**2. 货物运动学 (Payload Dynamics)**
货物受重力与绳索拉力共同作用，其加速度积分方程为：
$\ddot{p}_p = \mathbf{g} + \frac{T \mathbf{n}}{m_p}$

**3. 无人机受扰控制模型 (UAV Perturbed Kinematics)**
假设底层飞控完美跟踪期望指令速度 $\mathbf{v}_{cmd}$，但会受到货物的反作用拖拽力影响。实际执行速度包含受力扰动的积分项：
$\dot{p}_q(t) = \mathbf{v}_{cmd}(t) + \int_{0}^{\Delta t} \frac{-T \mathbf{n}}{m_q} d\tau$

## 🧠 强化学习环境设计 (MDP Design)

项目通过极其严密的数值缩放与奖励平衡，实现从起飞拉拽、平稳飞行、高难度穿梭到精准放置的端到端控制。

### 1. 状态空间 (Observation Space)

状态空间采用相对物理量，并经过严格的 $[-1.0, 1.0]$ 归一化处理。单智能体总观测维度为 **38 维**：

| 维度索引 | 物理含义 (Description) | 说明 |
| :---: | :--- | :---: |
| `0:3` | **无人机归一化速度** | $\mathbf{v}_q / v_{max}$ |
| `3:4` | **无人机归一化高度** | 映射至 $[-1.0, 1.0]$ 的绝对高度 |
| `4:7` | **货物相对位置** | $(\mathbf{p}_p - \mathbf{p}_q) / l_0$，隐式包含绳长与摆角信息 |
| `7:10` | **货物相对速度** | $(\mathbf{v}_p - \mathbf{v}_q)$，用于感知绳索拉扯与相对摆动状态 |
| `10:13` | **终点相对方向** | 货物指向终点的单位方向向量 |
| `13:14` | **终点归一化距离** | 当前货物距离目标点空间距离占环境最大对角线的比例 |
| `14:38` | **24 线 LiDAR 探测** | 360° 水平分布射线，返回最近障碍物的归一化触碰距离 |

### 2. 动作空间 (Action Space)

采用 **3 维连续动作空间**，直接控制无人机在惯性系下的三维期望速度分量：
* $\mathbf{a}_t \in [-1.0, 1.0]^3$：网络输出经过 $\text{Tanh}$ 激活，实际指令速度 $\mathbf{v}_{cmd} = \mathbf{a}_t \times v_{max}$。

### 3. 奖励函数重构与平衡 (Reward Function)

采用“正向势能引导 + 柔性姿态约束 + 稀疏成败截断”的混合结构：
$R_t = r_{step} + r_{dist} + r_{dir} + r_{smooth} + r_{alt} + r_{payload}$

* 🟢 **距离与方向势能 ($r_{dist}, r_{dir}$)**：鼓励智能体靠近目标且机头对准目标，提供持续的稠密正向激励。
* 🟡 **动作平滑与载荷惩罚 ($r_{smooth}, r_{payload}$)**：通过惩罚控制指令的二阶导数以及货物/无人机的相对速度（基于 L1 范数），强制压制绳索的剧烈摆动。
* 🟡 **动态高度惩罚 ($r_{alt}$)**：目标高度 $H_{target}$ 随无人机与终点水平距离动态衰减，完美引导无人机在巡航区保持安全高度，在终点区主动降落卸货。
* 🔴 **强截断终止 ($r_{goal}, r_{crash}$)**：到达终点奖励 $+2.0$，撞击障碍物惩罚 $-2.0$。所有单步基础奖励经严格调参保证在一个数量级，确保最终策略既不原地悬停，也不主动“碰瓷”。

## 🎓 三阶段课程学习 (Curriculum Learning)

鉴于“起飞拉扯瞬间易失控”和“带载荷穿梭窄缝”的双重极度困难，系统内置自动化调整的 `set_difficulty()` 课程接口：

* **第一阶段 (Level 1): 基础起降与平稳拉拽** (0 ~ 50万步)
  环境无任何障碍物。智能体专心学习如何抵抗非线性弹簧力的冲击，学会在不把货物甩上天的情况下，平稳地将其拖拽至终点。
* **第二阶段 (Level 2): 稀疏障碍物避障** (50万 ~ 150万步)
  环境中随机散布 10 个圆柱/方块障碍物。智能体开始将 LiDAR 感知数据与期望速度建立映射，学会在视距内绕开障碍物。
* **第三阶段 (Level 3): 密集极限穿梭大考** (150万步以后)
  障碍物数量提升至 25 个，且保留最小 3m 净通行间距。智能体将被极限压榨，涌现出在极其狭窄的缝隙中保持载荷绝对平稳的高级控制策略。
